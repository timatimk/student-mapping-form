# שאלון מיפוי תלמידים

פרויקט דף ווב אינטראקטיבי בעברית עבור הערכת מצב תלמידים לאחר חזרה מבית ספר וחופשת פסח.

## קבצים

- `index.html` - דף השאלון למילוי הנתונים.
- `dashboard.html` - דף הדשבורד להצגת הנתונים שנאספו.
- `style.css` - עיצוב RTL בעברית.
- `script.js` - לוגיקת הטופס, שמירת נתונים והצגת הודעות.
- `dashboard.js` - לוגיקת הדשבורד, סינון וייצוא נתונים.
- `data/students.json` - רשימת התלמידים לפי כיתה ומחנכת.

## שימוש

1. פתח את `index.html` בדפדפן כדי למלא שאלון עבור תלמיד.
2. עבור ל-`dashboard.html` כדי לראות את הנתונים שנשמרו, לסנן ולייצא ל-CSV.

> הנתונים נשמרים בדפדפן ב-`localStorage`. כדי לשתף דוחות או לאחסן נתונים במערכת מרכזית, ניתן להוריד את ה-CSV ולייבא אותו לכל מערכת.
