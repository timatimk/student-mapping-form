# שאלון מיפוי תלמידים - Branko Weiss

פרויקט דף ווב אינטראקטיבי בעברית עבור הערכת מצב תלמידים לאחר חזרה מבית ספר וחופשת פסח.
עיצוב מודרני בצבעי מכון ברנקו וייס.

## 📋 תכונות

- **שאלון מרובה שדות** - הערכת רמת תקשורת, רווחה נפשית, וכוחות/חוליות
- **דשבורד סטטיסטי** - סיכום נתונים עם גרפים ומספרים
- **סינון מתקדם** - סינון לפי כיתה, רמת תקשורת, ורמת רווחה
- **ייצוא CSV** - יצוא הנתונים לקובץ אקסל/CSV
- **עיצוב מודרני RTL** - ממשק ידידותי בעברית עם Branko Weiss branding
- **Responsive Design** - פועל בצורה מושלמת בנייד, טאבלט, ושולחן עבודה
- **Accessibility** - תמיכה בנגישות ומצבי עדיפות מערכת
- **Animations** - אנימציות חלקות וinteractions עמוקות

## 🚀 כיצד להשתמש

### 1️⃣ דף השאלון
```
https://timatimk.github.io/student-mapping-form/index.html
```
- בחר כיתה ומחנכת
- בחר שם תלמיד/ה
- הזן/בחר את:
  - **ציר התקשורת** (רמה 1-3)
  - **ציר הרווחה הנפשית** (ירוק/צהוב/אדום)
  - **זיהוי כוחות** - תיאור קצר
  - **נורות אדומות** - אם יש חוליות
  - **מצב משפחתי** - קונטקסט חברתי
  - **הערות נוספות** - פרטים נוספים
- לחץ "שמור תצפית"

### 2️⃣ דף הדשבורד
```
https://timatimk.github.io/student-mapping-form/dashboard.html
```
- צפה בסטטיסטיקות כוללות
- סנן לפי כיתה, תקשורת, או רווחה
- צפה בטבלת הנתונים שנשמרו
- לחץ "ייצא ל-CSV" להורדת דוח

## 💾 שמירת נתונים

הנתונים נשמרים **ב-localStorage של הדפדפן**:
- כל שיתוף חדש שומר לעמק המערכת
- הנתונים נשמרו גם אם סגרת את הדף
- אם מחקת את ה-cookies/cache, הנתונים יימחקו

**לאחסון ארוך טווח**, ייצא את הנתונים ל-CSV.
